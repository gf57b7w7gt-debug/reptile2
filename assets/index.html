Project idea

Safer reptile care with smart monitoring.

Reptiles need the right heat, humidity, and light. This project shows how a smart cage can use sensors and a camera to help owners notice problems earlier.

The website is designed for a school project presentation: the first view is simple, the media section uses real build photos, and the detailed view explains how each part works.

App pages

Gecko Monitor app

The app lets the owner check cage data, open the camera view, and review history. Each function has a clear purpose for reptile care.

Dashboard

Function: Shows the main cage information in one place, including temperature, humidity, analysis count, score, and live preview.

Camera

Function: Opens the camera page so the owner can watch the reptile and check if it is moving, hiding, or staying in one area too long.

History

Function: Saves previous data and analysis results so the owner can compare changes over time.

Total Analyses

Function: Counts how many times the system has checked the reptile or cage condition.

Avg Score

Function: Shows an average health or safety score based on the app analysis.

Temperature

Function: Shows the current cage temperature so the owner knows if the habitat is too hot or too cold.

Humidity

Function: Shows the moisture level in the cage and helps decide when humidity control is needed.

Live Preview and Start

Function: Displays the camera preview. The Start button begins the live check when the Raspberry Pi is connected.

Offline and Refresh

Function: Offline means the Raspberry Pi is not connected. The refresh button tries to reconnect and update the data.

Connect to Pi

Function: Reminds the owner to connect the Raspberry Pi before live data, camera preview, and analysis can work.
